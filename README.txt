Before publishing
1. Replace all placeholder tracking IDs:
   - G-XXXXXXXXXX
   - AW-XXXXXXXXXX
   - AW-XXXXXXXXXX/XXXXXXXXXXXXXXX
2. Update Calendly and email links if needed.
3. Replace the RS brand mark with your actual logo if desired.
4. If you use a real backend form, change the form action in index.html.

Tracking notes
- Base GA4 and Google Ads tags are included as placeholders.
- CTA click tracking placeholders use data-track attributes.
- thank-you.html includes a placeholder Google Ads conversion event.
- thank-you.html also includes a placeholder GA4 generate_lead event.
